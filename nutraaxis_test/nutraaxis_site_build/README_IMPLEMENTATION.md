# NutraAxis website page build

This package contains three completed page builds using the uploaded Archive templates as persistent header/footer/navigation wrappers.

## Files

- `rendered_pages/About_Us.html`
- `rendered_pages/Our_Science.html`
- `rendered_pages/Our_Quality.html`
- `main_snippets/about_us_main_snippet.html`
- `main_snippets/our_science_main_snippet.html`
- `main_snippets/our_quality_main_snippet.html`
- `NutraAxis_Page_Assets/` contains SVG and PNG graphics used by the pages.
- `mockups/` contains full-page single-image mockups for review.

## Implementation

Upload the `NutraAxis_Page_Assets` folder to the same path level as the HTML page, or update the image paths in the snippets.
Then replace only the content inside each page's `<main></main>` tag with the corresponding file from `main_snippets`.

## Brand system used

- Warm Teal: `#4A9B9B`
- Soft Coral: `#FF8A65`
- Cream: `#FAF8F3`
- Therapeutic accents: `#D4A574`, `#9B8FB9`, `#7EC4A3`, `#6BA3D9`

## Notes

The page copy is compliance-conscious structure/function language and avoids disease-treatment claims. Final content should still receive regulatory/legal review before publication.
